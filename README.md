# SHIT aka ShitLib
Super Helpful and Innovative Toolkit or 'SHIT' is a Java API which aims to make developing easier.

```xml
<dependencies>
    <!-- ... -->
    <dependency>
        <groupId>net.snackbag.shit</groupId>
        <artifactId>shit</artifactId>
        <version>1.0</version>
    </dependency>
</dependencies>
```
